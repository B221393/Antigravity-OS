## 🧠 蓄積システムの仕組み

### 1. 📚 Knowledge Cards（教養）

**保存先**: `knowledge_cards/outputs/cards/`

#### データ構造

```json
{
    "title": "カテゴリ - 日付",
    "genre": "就活・ES関連",
    "rarity": "Legendary",  // スコアで自動判定
    "timestamp": "2026-01-16 19:00:00",
    "content": "最初の500文字",
    "source": "Text Router - 自動仕分け",
    "full_text": "完全なテキスト",
    "routing_scores": {"就活・ES関連": 5.0, ...}
}
```

#### レアリティ判定

- **Legendary**: スコア 5.0以上（超高マッチ）
- **Epic**: スコア 3.0-4.9（高マッチ）
- **Rare**: スコア 2.0-2.9（中マッチ）
- **Uncommon**: スコア 1.0-1.9（低マッチ）
- **Common**: スコア 1.0未満（キーワードなし）

### 2. 🧬 Ego Memory（自己記憶）

**保存先**: `memory/data/`

#### データ構造

```json
{
    "type": "thought",
    "category": "就活・ES関連",
    "timestamp": "2026-01-16T19:00:00",
    "content": "完全なテキスト",
    "context": {
        "source": "text_router",
        "auto_captured": true,
        "routed_category": "就活・ES関連"
    },
    "metadata": {
        "length": 500,
        "has_keywords": true
    }
}
```

## 🔄 連携フロー

### パターン1: ES作成の完全フロー

```
1. Text Routerにテキスト入力
   「講談社のESを書きたい。編集者として面白い本を作りたい」
   
2. 自動仕分け
   → カテゴリ: 就活・ES関連
   → スコア: 3.0（"ES", "編集者"がマッチ）
   
3. 知識蓄積 ✨
   📚 Knowledge Card作成
      - レアリティ: Epic
      - ジャンル: 就活・ES関連
      - ファイル: 就活・ES関連_2026-01-16_190000.kcard
   
   🧠 Ego Memory作成
      - タイプ: thought

   → クイックランチでES作成開始
```

3. 自動仕分け

4. 知識として蓄積
   📚 あなたの就活に関する思考が記録される
   🧠 エゴが「あなたの不安」を記憶する

## 💡 Tips

### 1. すべてが記録される

- Text Routerに入力したテキストは**すべて**知識として蓄積されます
- 後から見返せるように、思いついたことはどんどん入力しましょう

### 2. スコアが高いほど価値が高い

- キーワードが多くマッチするほどレアリティが上がります
- レアリティが高い = より専門的で価値の高い知識

### 3. エゴが成長する

- Ego Memoryに蓄積された思考は、Ego Personaで「あなたの分身」を作る材料になります
- 継続的に使うことで、AIがあなたの思考パターンを学習します

### 4. 知識カードで復習

- Knowledge Cardsアプリで、過去の知識を振り返り可能
- レアリティでソートして、重要な知識を優先的に復習
