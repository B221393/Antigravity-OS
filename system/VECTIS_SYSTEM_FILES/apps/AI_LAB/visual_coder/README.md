# Visual Coder

Visual Coder は、ビジュアルなフローチャートベースのロジック、従来のコード生成、3D可視化、そしてAIアシスト開発の間の橋渡しをするために設計された、革新的なウェブベースのビジュアルプログラミング環境です。Next.js、React Flow、Three.js、Monaco Editor を用いて構築されており、ユーザーが直感的かつ視覚的な方法でソフトウェアロジックを作成・理解することを可能にします。

## 主な機能

* **ノードベースプログラミング**: 多様なノード（例: Print、変数、If/Else、ループ、関数、3Dオブジェクト）をドラッグ＆ドロップでキャンバスに配置し、接続することでプログラムロジックを構築します。
* **多言語コード生成**: 視覚的なフローチャートをPython、Rust、Cの実行可能なコードに自動的にトランスパイルし、リアルタイムのコードエディタで確認できます。
* **統合された3D可視化**: キューブや球体のような3Dオブジェクトをロジックに追加し、アプリケーションの3Dキャンバス内で直接可視化することで、コードの効果をユニークな「3Dウォークスルー」体験として提供します。
* **AIアシスト開発**: AIを活用した高度な機能：
  * **AIコード生成**: 自然言語のプロンプトに基づいて、特定のノードのコードスニペットを生成します。
  * **コードからビジュアルへの変換**: 既存のPythonコードをインタラクティブなノードグラフに変換します。
  * **AIリファクタリング**: AIによる提案で生成されたコードを最適化・改善します。
* **プリセットコードブロック**: 定義済みのノードセットを使用して、一般的なプログラミングパターンやロジックを素早く挿入します。
* **インタラクティブコードエディタ**: 統合されたMonaco Editorで生成されたコードを確認でき、コピーやダウンロードのオプションも利用できます。

## セットアップ

Visual Coder開発サーバーをセットアップし、実行するには以下の手順に従ってください。

1. **プロジェクトディレクトリへ移動**:

    ```bash
    cd VECTIS_SYSTEM_FILES/apps/AI_LAB/visual_coder
    ```

2. **依存関係のインストール**:

    ```bash
    npm install
    # または yarn install
    # または pnpm install
    # または bun install
    ```

3. **開発サーバーの実行**:

    ```bash
    npm run dev
    # または yarn dev
    # または pnpm dev
    # または bun dev
    ```

4. ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスし、アプリケーションを開きます。

## 使用方法

アプリケーションが実行されたら：

1. **ノードの追加**: 左側のサイドバーを使用して、様々な種類のノード（例: "Print"、"Variable"、"If / Else"、"Cube"、"Sphere"）をキャンバスにドラッグ＆ドロップします。
2. **ノードの接続**: ノードのポート間を接続し、プログラムロジックのフローを定義します。
3. **ノードの設定**: (今後の改善点: 現在はプロンプトを使用して設定します。専用のプロパティパネルが計画されています。)
4. **コード生成の確認**: 右側のパネルに、視覚的なフローチャートに基づいてリアルタイムで生成されたコードが表示されます。ターゲット言語としてPython、Rust、Cを選択できます。
5. **3Dとのインタラクション**: 「Cube」や「Sphere」のようなノードは、背景の3Dキャンバスに表示されます。
6. **AI機能の活用**: 「AI Mode」を切り替えることで、AIを活用したノードのコード生成、リファクタリング、Pythonコードからビジュアルノードへの変換を利用できます。
