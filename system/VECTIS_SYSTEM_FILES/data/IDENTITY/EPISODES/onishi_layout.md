# ⌨️ Episode: The Cybernetic Interface (Onishi Layout)

## Key Concept

**"The Input Bottleneck Problem" (思考速度 > 入力速度)**

## Story

現代のエンジニアリングにおける最大のボトルネックは、CPUでも通信速度でもなく、「人間の指」だ。
私は、思考がキーボード入力の遅さによって阻害されることに我慢ならなかった。

そこで、標準的なQWERTY配列を捨て、日本語の母音・子音統計に基づき最適化された「大西配列」への移行を決断した。
これは単なる「設定変更」ではない。自分の**身体的インターフェース（レイヤー0）を再プログラム**する行為だ。
数ヶ月の再学習を経て、今は「脳神経が直接IDEに繋がっている」かのような直結感を得ている。

## Reusable Asset (ES/Interview)

- **Technical Angle**: HCI (Human-Computer Interaction) への深い理解。
- **Personality**: 合理性のために、慣れ親しんだ習慣さえも破壊・再構築できる「適応力」。
