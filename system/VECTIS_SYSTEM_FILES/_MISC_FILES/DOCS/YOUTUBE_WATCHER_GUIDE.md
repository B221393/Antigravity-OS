## 🔧 エラー対策

### 1. HTTP 404エラー（社會チャンネル）

**原因**: チャンネルID `UCsNQlg8pVqp6ZWUM8WmTKNw` が無効

**解決策**:

```json
// channels.jsonを編集
{
    "name": "社會",
    "channel_id": "正しいID",  // ← YouTubeで確認
    "enabled": false  // または無効化
}
```

**正しいIDの見つけ方**:

1. YouTubeでチャンネルページを開く
2. URLから `UC...` 部分をコピー
   - 例: `https://www.youtube.com/channel/UC8yHePe_RgUBE-waRWy6olw`
   - → `UC8yHePe_RgUBE-waRWy6olw`
