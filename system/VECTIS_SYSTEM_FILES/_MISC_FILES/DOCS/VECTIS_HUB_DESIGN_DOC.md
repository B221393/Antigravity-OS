
# 📄 VECTIS HUB System Specification & Design Document (v3.1 Knowledge Galaxy)

## 1. System Overview

**VECTIS HUB** is the central command station for the user's personalized OS. It acts as a launcher and information aggregator for various sub-applications (Job Hunting, Learning, Tools).

### 1.1 Core Philosophy

- **"Personal Station"**: A unified interface that feels like a professional broadcasting station or a space command center.
- **Novelty Discovery**: Autonomous agents (YouTube Watcher) constantly fetch and analyze external information to present "novel" ideas to the user.
- **Aesthetic**: "Retro-Future / Space / Sakuma Style". Bold typography, glassmorphism, and a sense of scale (Earth from Space).

## 2. Architecture

### 2.1 Technology Stack

- **Frontend/Dashboard**: Streamlit (Python)
- **Sub-Apps**:
  - Streamlit (most apps)
  - React/Vite (YouTube Summarizer)
- **Backend/Logic**: Python (standard library + custom modules)
- **Data Storage**: JSON (auto-generated by watchers)

### 2.2 Navigation Structure

- **Global Navigation (`modules/nav.py`)**: A shared sidebar component used across all Streamlit apps.
- **Port Management**:
  - `8501`: Dashboard (HUB)
  - `8502`: TOEIC Mastery
  - `8511`: YouTube Summarizer (React)
  - `8512`: Memory 3D (Knowledge Galaxy)
  - `8517`: Job HQ
  - etc.

## 3. Design System (v3.0 "Space Edition")

### 3.1 Visual Concept

- **Background**: High-resolution image of Earth from Space, representing a global/macro perspective.
- **Container**: Glassmorphism (translucent dark panels with light borders).
- **Typography**:
  - Headers: `Oswald` (Bold, Condensed, Uppercase) - "Producer Style"
  - Body: `Noto Sans JP` (Readable, Clean)
  - Code/Data: `Share Tech Mono` (Cybernetic feel)
- **Colors**:
  - **Cyan (`#00E5FF`)**: Primary accent (Intelligence, Signal)
  - **White (`#FFFFFF`)**: Text, Stars
  - **Magenta (`#FF3366`)**: Alerts, Emphasis
  - **Recency Gradient (Blue → Red)**: Knowledge Galaxy node colors

### 3.2 Key Components (`modules/style.py`)

- `apply_vectis_style()`: Injects global CSS for the Space theme.
- `get_station_header()`: Renders the standardized "ON AIR" header.

## 4. Key Features

### 4.1 YouTube Novelty Analysis

- **Watcher**: `AUTO_YOUTUBE/RUN_WATCHER.bat` scans subscribed channels and analyzes video transcripts for "Novelty".
- **Display**: The Dashboard reads `novelty_scores_v2.json` and displays high-scoring videos.
- **Viewer**: Users can click "Read Summary" to view the full markdown summary generated by Gemini.

### 4.2 Application Launcher

- A grid view of all available VECTIS applications.
- Direct linking to localhost ports.
- Categorized into "Job Hunting", "Learning", "Tools".

### 4.3 Knowledge Galaxy (3D Visualization) ✨ NEW

- **Embedded 3D Map**: Interactive Plotly Scatter3D embedded in the Dashboard's YouTube Analysis tab.
- **Node Display**:
  - **Sphere Size**: Adjustable via slider (default: small spheres for clarity).
  - **Color by Recency**: Blue = older content, Red = newer content.
  - **Random Sampling**: Displays a subset (default 30) of nodes to avoid clutter.
- **Relationship Lines**:
  - Lines connect nodes with keyword similarity > 10%.
  - **Thickness** represents relationship strength (more shared keywords = thicker line).
- **Node Detail Panel (Twitter-style)**:
  - Select a node from dropdown.
  - Expander shows: Summary (first 800 chars), Source URL button, and last-modified date.
  - No page navigation required.
- **Fullscreen**: Button links to dedicated Memory 3D app on port 8512.

## 5. Deployment / Start-up

- **All Apps**: Run `START_ALL_APPS.bat`
- **Stop All**: Run `STOP_ALL_APPS.bat`
- **YouTube Watcher**: Run `RUN_WATCHER.bat` (in `VECTIS_SYSTEM_FILES/AUTO_YOUTUBE/`)
