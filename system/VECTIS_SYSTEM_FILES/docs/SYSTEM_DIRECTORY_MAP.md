# VECTIS プロフェッショナル・ワークステーション ディレクトリ構成仕様書

## 1. システム概要

本システム `VECTIS_SYSTEM_FILES` は、多種多様なアプリケーション、AIモジュール、メディア管理ツールを統合した複合ワークステーション環境である。
すべてのアプリケーションは `apps` ディレクトリ配下に配置され、その役割に応じて4つの主要カテゴリーに分類されている。

**ルートパス:** `VECTIS_SYSTEM_FILES/apps/`

---

## 3. データ構造指針

### アプリケーション開発ルール

1. **独立性**: 原則として各アプリは独立したディレクトリを持ち、自身の `app.py` で完結または起動する。
2. **データ連携**: アプリ間のデータ共有は `apps/SYSTEM/data_hub` または共通のJSONファイルを介して行う。
3. **UI統一**: `modules/style.py` などを利用し、VECTIS共通のデザイン言語（ダークテーマ、ネオンアクセント）を適用する。

### 仮想環境

- **パス**: `.venv/`
- 全てのPythonアプリケーションはこの共通の仮想環境下で実行されることを前提とする。
- 起動コマンド例: `..\..\.venv\Scripts\streamlit run app.py`
