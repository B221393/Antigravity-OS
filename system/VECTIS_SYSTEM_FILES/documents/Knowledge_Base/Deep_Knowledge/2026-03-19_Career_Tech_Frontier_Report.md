# 2026-03-19 Career-Tech Frontier Report: 巨大インフラのAI・Rust革命

本レポートは、2025年〜2026年にかけて発表されたオープンな学術報告・技術レポートに基づき、志望企業（日本郵船、JR、三菱重工、NTT）が現在直面している技術的変革をまとめたものである。

---

## 1. 日本郵船 (NYK): 自律運航船「DFFAS+」と2026年商用化ロードマップ
- **出典概要**: Reliability Engineering & System Safety (2025) および MEGURI2040 Stage 2 報告書。
- **核心技術**:
    - **APExS-auto**: AIが航路上の他船を検知し、衝突回避計画を自動立案するが、最終決定は人間（船員）が行う「協調型自律システム」。
    - **ROC (陸上支援センター)**: 2026年3月に竣工する新型自動車船を皮切りに、陸上から複数の無人船を遠隔監視する体制。
- **面白さのポイント**:
    - 単なる「自動運転」ではなく、JAXAのロケット回収技術（自動船位保持）を民間に転用するという、宇宙×海運のクロスオーバーが起きている点。
- **面接での武器**:
    - 「2026年の商用化に向けた実証トライアルにおいて、自律システムの安全性評価やエッジ側でのデータ処理の知見を活かしたい」と語る。

## 2. JR東日本: 「鉄道版生成AI」とスマートメンテナンスの極致
- **出典概要**: JR East Technical Review No.74 (2025予定) および「変革2027」進捗報告。
- **核心技術**:
    - **信号通信設備復旧AI**: 2025年度導入。RAG（検索拡張生成）を用いて、過去の故障データと社内規定を瞬時に照合し、最適な復旧手順を現場に提示。
    - **ドローン画像解析**: 山手線などで、パンタグラフや架線の異常をディープラーニングで自動検知。
- **面白さのポイント**:
    - 「ベテランの経験」という暗黙知を、生成AIによって「全社員が使える形式知」に変換し、復旧時間を半分（50%削減）にするという破壊的効率化。
- **面接での武器**:
    - 「鉄道という止まってはいけないインフラに、生成AIを安全に（ハルシネーションを抑えて）実装する取り組みに、自身のプログラミングスキルで貢献したい」と語る。

## 3. 三菱重工 (MHI) / 産業界: Rustによるサイバーレジリエンス革命
- **出典概要**: Safety-Critical Rust Consortium 2025 ガイドライン。
- **核心技術**:
    - **Ferrocene**: ISO 26262 認証を取得したRustツールチェーン。
    - **メモリ安全性**: C++で頻発していたバッファオーバーフロー等の脆弱性を、Rustの「所有権」概念でコンパイル時に100%排除。
- **面白さのポイント**:
    - セキュリティインシデントを47%削減しつつ、リアルタイム性を維持するという、重工・防衛分野での「信頼性の新基準」。
- **面接での武器**:
    - 「安全性と速度が両立しなければならない重工システムにおいて、Rustのようなメモリ安全な言語を自律システムに導入し、堅牢な基盤を作ることにパッションがある」と語る。
