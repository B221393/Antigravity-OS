# 企業分析レポート: 三菱重工 (MHI)
- **初任給 (学部卒)**: 月給 255,000円
- **初任給 (修士了)**: 月給 285,000円
- **主要な福利厚生**: 借上社宅、保養所、防衛手当
- **技術投資の方向性**: AI Factory、水素エネルギー

## 戦略的考察
1. **経済的価値**: 初任給は横並びだが、借上社宅、保養所、防衛手当による実質的な可処分所得の差に注目。
2. **技術的価値**: AI Factory、水素エネルギーに予算が割かれており、エンジニアとして挑戦的な環境。
3. **キャリアパス**: 高い初期給与だけでなく、技術投資が活発な分野（自律化等）での市場価値向上を目指すべき。
