* **概要:** 就職活動における膨大な情報の整理と分析のボトルネックを解消するため、「自分専用の戦略的参謀AI」として統合システム（思考OS）を個人開発。
* **技術要素:**
  * Pythonによるスクレイピング（情報収集）
  * Three.jsによる可視化
  * 生成AIの統合
  * Webアプリ開発

* **学び・成果:**
  * 「非効率を技術で打破する」課題解決力。
  * 「目的に向かって最適なツールを自ら作り出す力」。
  * 「散在する情報を整理・構造化し、自ら課題を設定して解決するプロセス」に情熱を感じた。
