# 認知バイアス大全 (ビジネス意思決定編)

意思決定の質を上げるため、そして面接で「メタ認知」をアピールするために。

---

## 1. 主要な認知バイアス一覧

| バイアス | 説明 | ビジネス例 |
| :--- | :--- | :--- |
| **確証バイアス** | 自分の信念を支持する情報だけ重視 | 新規事業の成功を確信→リスクを軽視 |
| **アンカリング効果** | 最初の情報に引きずられる | 交渉で最初の提示価格に縛られる |
| **利用可能性ヒューリスティック** | 思い出しやすい情報で判断 | 成功体験に引っ張られすぎる |
| **現状維持バイアス** | 変化を避け現状を好む | 新技術への対応遅れ |
| **サンクコスト効果** | 既に投じたコストを惜しみ撤退できない | 赤字プロジェクトを「ここまで来たから」と続ける |
| **フレーミング効果** | 表現の仕方で判断が変わる | 「成功率80%」vs「失敗率20%」 |
| **バンドワゴン効果** | 多数派に従いたがる | トレンドに追随、本質を見誤る |
| **正常性バイアス** | 危険を過小評価 | 「まだ大丈夫」で対応が遅れる |
| **楽観バイアス** | 自分だけはうまくいくと過信 | リスク軽視、資金計画の甘さ |
| **後知恵バイアス** | 結果から「やはりそうだと思った」と錯覚 | 失敗分析の客観性喪失 |
| **保有効果** | 所有物を手放す際に過大評価 | 自社技術への過剰な愛着 |

---

## 2. バイアスを克服する方法

1. **チェックリスト**: 意思決定前にバイアスリストを確認。
2. **悪魔の代弁者**: 反対意見を意図的に出す役を設ける。
3. **プレモーテム**: 「失敗した未来」を想定して逆算。
4. **データドリブン**: 直感だけでなく、データで検証。
5. **多様な視点**: 異なるバックグラウンドのメンバーで議論。

---

## 3. 面接での活用

### 質問例: 「あなたの弱みは？」
>
> 「私は確証バイアスに陥りやすい傾向があります。自分のアイデアを支持する情報ばかりを集めてしまうことがあります。そのため、意識的に反対意見を求めたり、プレモーテム（失敗を想定した逆算）を行う習慣をつけています。」

### 質問例: 「困難を乗り越えた経験は？」
>
> 「プロジェクトで一度始めたアプローチに固執してしまい、サンクコスト効果に陥りかけました。しかし、冷静にゼロベースで再評価し、撤退と方針転換を決断しました。」

---

## 4. EGO的意義

**メタ認知こそ知性**。自分の思考の癖を自覚し、補正できることが、本当の「頭の良さ」。
バイアスを知ることは、**自分自身の取扱説明書**を持つことである。
