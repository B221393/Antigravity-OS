 VECTIS (AI秘書) - ユーザーマニュアル
==============================================================================

【1. 起動方法】
フォルダ内の `run_vectis_apps.bat` をダブルクリックしてください。
以下のメニューが表示されます：

[1] Launch Secretary (Local Voice/Chat)
    - PCのマイクとスピーカーを使って、音声で直接会話します。
    - または、キーボードでチャット入力します。

[2] Launch Knowledge Card Viewer
    - これまでに作成した「知識カード」を閲覧・検索するアプリを開きます。

[3] Launch Mobile Bridge Server
    - スマホから遠隔操作するためのサーバーを起動します。
    - 起動後、画面に表示される URL (例: http://192.168.1.5:8000) を
      スマホのブラウザに入力してください。

[4] Open Output Folder
    - 生成されたファイル（要約、カード、ログなど）が保存されているフォルダを開きます。

------------------------------------------------------------------------------

【2. スマホ連携（Mobile Bridge）の使い方】
1. メニューで [3] を選択。
2. 黒い画面（コンソール）に「Mobile IP: ...」や「URL」が表示されます。
3. スマホの Chrome や Safari でその URL にアクセスします。
   ※ Wi-Fi に接続している必要があります。
4. スマホ画面の「VOICE INPUT」ボタンを押して話しかけるか、
   テキストボックスに入力して「EXECUTE」を押します。

------------------------------------------------------------------------------

【3. 主なコマンド例】
Vectisはあなたの言葉を理解して自動で判断しますが、以下のキーワードを使うと確実です。

■ 調査・検索
「○○について調べて」
「今日のAIニュースを教えて」
 -> ネット検索を行い、要約して教えてくれます。

■ ホロデッキ（アイデア出し）
「○○についてホロデッキでアイデア出して」
「就活の自己分析をホロデッキでやって」
 -> 5つの視点（情熱、市場、スキル、未来、意外性）から多角的に分析し、
    結果を保存＆カード化します。

■ 知識カード作成
「○○をカードにして」
 -> 調べた内容や指定したテーマを、保存用の「知識カード」にします。

■ PC操作（実験的機能）
「ブラウザを開いて」「ここをクリックして」
 -> 画面を見て操作を行おうとします（Voice/Visionモード時）。

------------------------------------------------------------------------------

【4. トラブルシューティング】

Q. キーボード配列がおかしい（大西配列になっている）
A. `khd.bat` を実行するか、コマンドプロンプトで `khd` と入力してください。
   即座に通常の QWERTY 配列に戻ります。
   （または `Ctrl + Alt + Q` を押すとAHKスクリプトが終了します）

Q. スマホから繋がらない
A. PCとスマホが同じ Wi-Fi に繋がっているか確認してください。
   「localhost」ではなく「192.168...」のIPアドレスを使ってください。

Q. エラーが出る
A. `outputs/logs/activity_log.md` にログが残っています。
