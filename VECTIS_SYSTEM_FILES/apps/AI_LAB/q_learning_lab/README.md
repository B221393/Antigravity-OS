
# 🧠 Q-Learning Lab (Q学習実験室)

このラボは、強化学習（Reinforcement Learning）の基礎おなる「Q学習」を、シンプルな迷路（GridWorld）を通じて体験・理解するために設計されています。

## 📂 構成

- **envs/maze_env.py**: 4x4の迷路環境。スタート(S)、ゴール(G)、穴(H)が存在します。
- **agents/q_agent.py**: 試行錯誤を通じて「最適な行動価値(Q値)」を学習するエージェント。カンニングペーパー（Qテーブル）を作成します。
- **run_maze.py**: 学習を実行し、結果を可視化するメインスクリプト。

## 🚀 実行方法

以下のコマンドで学習をお試しください。

```bash
python run_maze.py
```

約2000回の試行（エピソード）を行い、最初はデタラメに動いていたAIが、徐々に「穴を避けてゴールする最短ルート」を見つけ出す様子が観察できます。

## 🚗 「車の運転」への応用について（How to Scale）

このラボではシンプルな「マス目」を使っていますが、この仕組みはそのまま自動運転のような複雑なタスクへ発展させることができます。

1. **DQN (Deep Q-Network) への進化**:
    - **現在**: 「マス目」ごとの点数を表（Qテーブル）にメモしている。
    - **未来**: 「カメラ映像（ピクセル）」を入力し、ニューラルネットワークが「ハンドルの角度」ごとの点数を予測する。
    - マス目が無限にあるような複雑な状況（道路状況）でも、Deep Learningを使えば「似たような状況」から推測して運転できるようになります。

2. **Actionの連続化**:
    - **現在**: 上・下・右・左 の4択（離散値）。
    - **未来**: アクセル開度 0%〜100%、ハンドル角度 -45度〜+45度 のような滑らかな操作（連続値）。これには「DDPG」や「PPO」といった、Q学習の発展系アルゴリズムが使われます。

このラボで「報酬（Reward）による学習」の感覚を掴んだら、次は物理演算ライブラリ（PyBulletやUnity ML-Agents）を使った自動運転シミュレーションに挑戦するのがおすすめです！
