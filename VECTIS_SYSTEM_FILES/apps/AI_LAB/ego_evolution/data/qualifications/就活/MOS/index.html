MOS 対策

Word

よく使うショートカットキーを覚えて効率を上げましょう。

  • Ctrl + C: 選択範囲をコピー
  • Ctrl + V: 貼り付け
  • Ctrl + X: 切り取り
  • Ctrl + S: 上書き保存
  • Ctrl + B: 太字
  • Ctrl + I: 斜体

Excel

基本的な関数をマスターしましょう。

  • SUM: 合計を求める (例: =SUM(A1:A10))
  • AVERAGE: 平均を求める (例: =AVERAGE(A1:A10))
  • IF: 条件によって表示を変える (例: =IF(A1>80, "合格", "不合格"))
  • COUNT: 数値データの個数を数える (例: =COUNT(A1:A10))

PowerPoint

見やすいプレゼンテーション作成のコツです。

  • 1スライド1メッセージを心がける
  • 文字を詰め込みすぎない
  • グラフや図を効果的に使う
  • 統一感のあるデザイン（色、フォント）を意識する
