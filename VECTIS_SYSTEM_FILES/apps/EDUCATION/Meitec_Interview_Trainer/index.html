📊 Google Spreadsheet 連携

Google Apps Script (GAS) を使って、スプレッドシートから質問データを取得できます。

設定手順:

  1. Google Spreadsheet を作成し、下記フォーマットで入力
  2. 「拡張機能」→「Apps Script」を開く
  3. 下記の GAS コードを貼り付け
  4. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」として公開
  5. 生成された URL をここに貼り付け

スプレッドシートの形式:

質問テキスト ヒント 目安秒数
自己紹介を... 軸を30秒で 60

GAS コード（コピペ用）:

function doGet() {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName("質問");
  const data = sheet.getDataRange().getValues();
  const headers = data[0];
  const questions = [];
  for (let i = 1; i < data.length; i++) {
    if (data[i][0]) {
      questions.push({
        text: data[i][0],
        hint: data[i][1] || "",
        ideal: parseInt(data[i][2]) || 60
      });
    }
  }
  return ContentService
    .createTextOutput(JSON.stringify({ questions }))
    .setMimeType(ContentService.MimeType.JSON);
}
